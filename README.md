# 🍷 Wine Quality Analysis & Regression Project

## 📌 Objective
Analyze wine dataset and predict wine quality using Linear Regression.

## 📊 Dataset
Wine Quality Dataset from UCI Machine Learning Repository.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📈 Model
Linear Regression

## 🔍 Key Insights
- Alcohol positively affects quality
- Volatile acidity negatively affects quality
- Model achieved moderate R² score
